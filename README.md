About sseclient-py
================================

Home: https://github.com/snowflakedb/sseclient-py

Package license: Apache-2.0

Feedstock license: [BSD-3-Clause](https://github.com/AnacondaRecipes/sseclient-py-feedstock/blob/main/LICENSE)

Summary: Snowflake machine learning library.

Development: https://github.com/snowflakedb/sseclient-py

Documentation: https://docs.snowflake.com/developer-guide/snowflake-ml/overview

The machine learning client library that is used for interacting with Snowflake to build machine learning solutions. Functionalities include model uploading, training, feature engineering, etc.


Installing sseclient-py
==================

Installing `sseclient-py` from the main channel can be achieved by:

```
conda install sseclient-py
```

It is possible to list all of the versions of `sseclient-py` available on your platform with `conda`:

```
conda search sseclient-py
```
